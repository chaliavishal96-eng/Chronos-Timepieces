# Chronos Time Pieces

An independent online marketplace for pre-owned luxury watches.
Best price by bank wire — buyers inquire, no card checkout.

## Run it locally

Open the folder in VS Code and use the **Live Server** extension
(right-click `index.html` → "Open with Live Server"), or just
double-click `index.html` to open it in your browser.

## Files

- `index.html` — page structure
- `styles.css` — all styling
- `script.js` — interactivity (filters, inquiry form, etc.)

## Deploy

Push to GitHub, then enable **GitHub Pages** (Settings → Pages → Branch: `main`),
or connect the repo to **Vercel** or **Netlify** for automatic deploys.
